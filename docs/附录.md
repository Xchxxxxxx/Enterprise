# 附录

## A. 框架包清单

| 包名 | 包含内容 | 依赖 |
|------|---------|------|
| `EfCore.Enterprise.Shared` | 枚举、模型、异常、特性、工具类、`[Injectable]`、`ApiResult` | 无 |
| `EfCore.Enterprise.Domain` | 实体基类、事件基类、领域服务接口、仓储接口、工作单元接口 | `EfCore.Enterprise.Shared` |
| `EfCore.Enterprise.Infrastructure` | DbContext、仓储实现、EF Core 拦截器、**8个中间件**、**功能拆分扩展**（Swagger/JWT/OpenTelemetry/Serilog/Cors）、缓存、锁、后台任务 | `EfCore.Enterprise.Domain` |
| `EfCore.Enterprise.Application` | `BaseService`、`BaseProfile`、`CrudAppService`、**`BaseApiController`**、**`CrudController`**、MediatR 集成 | `EfCore.Enterprise.Domain`、`EfCore.Enterprise.Shared`、`EfCore.Enterprise.Infrastructure` |
| `EfCore.Enterprise.Cli` | 命令行工具：代码生成、项目创建、迁移管理 | .NET 8 CLI |
| `EfCore.Enterprise.Templates` | 项目模板：`dotnet new ef-enterprise` | .NET 8 Templates |

> **架构变更**：不再包含 `EfCore.Enterprise` 元包。原 Presentation 层功能已拆分合并到 Infrastructure 和 Application 包中。

### E.3 各配置节详细说明

| 配置节 | 必填 | 默认值 | 说明 |
|--------|:----:|--------|------|
| `ConnectionStrings:DefaultConnection` | ✅ | — | 数据库连接字符串。框架使用 MySQL，也可替换为 SQL Server / PostgreSQL |
| `Jwt:SecretKey` | ✅ | — | JWT 签名密钥，**至少 32 个字符**。生产环境务必用强随机字符串，切勿提交到代码仓库 |
| `Jwt:Issuer` | ✅ | — | JWT 签发者标识，任意字符串，如 `YourProject` |
| `Jwt:Audience` | ✅ | — | JWT 受众标识，任意字符串，如 `YourProject` |
| `Jwt:AccessTokenExpirationMinutes` | ⭕ | `30` | AccessToken 过期时间（分钟）。过期后需要用 RefreshToken 续期 |
| `Jwt:RefreshTokenExpirationDays` | ⭕ | `7` | RefreshToken 过期时间（天）。过期后需要重新登录 |
| `Snowflake:WorkerId` | ⭕ | `1` | 雪花ID 工作器 ID（0-31）。**集群环境每个节点必须唯一** |
| `Snowflake:DatacenterId` | ⭕ | `1` | 雪花ID 数据中心 ID（0-31）。跨机房部署时区分 |
| `Redis:Connection` | ⭕ | — | Redis 连接字符串。不填则使用内存缓存降级 |
| `Cors:Origins` | ⭕ | — | 允许跨域的前端域名列表。不填默认允许所有来源 |
| `OpenTelemetry:Endpoint` | ⭕ | — | OpenTelemetry Collector 地址（gRPC）。不填则输出到控制台 |
| `Monitoring:SlowSqlThresholdMs` | ⭕ | `200` | 慢 SQL 告警阈值（毫秒）。超过此值会在日志中 WARN |
| `Monitoring:NPlusOneWarningThreshold` | ⭕ | `10` | N+1 查询检测阈值。同一请求内 SQL 查询次数超过此值会 WARN |
| `Serilog` | ⭕ | — | Serilog 结构化日志配置。框架通过 `ReadFrom.Configuration` 读取，支持所有 Serilog 配置项 |

## F. Program.cs 精简版

```csharp
using EfCore.Enterprise.Application.Extensions;
using EfCore.Enterprise.Infrastructure.Configuration;

var builder = WebApplication.CreateBuilder(args);

builder.AddEfCoreSerilog();

// 一键注册（Swagger + JWT + OpenTelemetry + Cors + 基础设施 + 应用层）
builder.Services.AddEfCoreEnterprise(builder.Configuration);

// 自动注入（[Injectable] + FluentValidation + MediatR + AutoMapper）
builder.Services.AddEfCoreAutoInject(assemblies);

var app = builder.Build();

// 一键配置中间件管道
app.UseEfCorePipeline(app.Environment.IsDevelopment());

app.MapHealthChecks("/health");
app.Run();
```

## G. 常用命令

```bash
# 创建项目
dotnet new ef-enterprise -n MyApp -o MyApp

# 添加迁移
dotnet ef migrations add InitialCreate -p MyApp/05-Infrastructure -s MyApp/06-Presentation

# 更新数据库
dotnet ef database update -p MyApp/05-Infrastructure -s MyApp/06-Presentation

# 生成 NuGet 包
.\pack-all.ps1

# 注册本地 NuGet 源（只需一次，任意路径执行）
& "d:\自建\项目\ef\setup-local-nuget.ps1"

# 安装项目模板
dotnet new install EfCore.Enterprise.Templates
```

## J. 快速参考卡片

| 我想... | 做法 |
|---------|------|
| 创建实体 | 继承 `BaseFullEntity`，属性 `private set` |

| 分页查询 | `_repo.GetPagedAsync(predicate, request)` |
| 软删除 | `_repo.SoftDeleteAsync(id)` |
| 恢复数据 | `_repo.RestoreAsync(id)` |
| 事务 | `_uow.BeginTransactionAsync()` + `CommitTransactionAsync()` |
| 领域事件 | 实体中 `AddDomainEvent()`，Handler 实现 `INotificationHandler<T>` |
| 缓存 | `_cache.GetOrSetAsync(key, factory, ttl)` |
| 分布式锁 | `await using (await _cache.LockAsync(key, ttl))` |
| 后台任务 | `_jobService.EnqueueAsync(() => ...)` |
| 定时任务 | `_jobService.AddRecurring(name, action, cron)` |
| 权限控制 | Controller 上加 `[Permission("...")]` |
| 限流 | Action 上加 `[RateLimit(n, seconds)]` |
| 幂等 | 请求头加 `Idempotency-Key` |
| 熔断 | `_resilience.ExecuteAsync("name", func, fallback)` |
| 参数验证 | 创建 `AbstractValidator<T>` |
| DI 注册 | 类上加 `[Injectable]` |
| AutoMapper | 继承 `BaseProfile`，覆写 `Configure()` |
| 零代码 CRUD | 继承 `CrudAppService<TEntity, TDto, TCreateDto, TUpdateDto>` |
| 树形结构 | 继承 `BaseTreeEntity<T>` |
| 合规归档 | 继承 `BaseComplianceEntity` |
| 统一响应 | `ApiResult<T>.Ok(data)` / `ApiResult.Fail(msg)` |
| 分页响应 | `PagedResult<T>.Ok(items, total, page, size)` |
| 抛出异常 | `throw new BusinessException("...")` |
| JWT 认证 | 注入 `IJwtService` |
| 健康检查 | `GET /health` |
| 性能指标 | `GET /metrics` |
| 数据库迁移 | `dotnet ef migrations add ...` |
| 本地打包 | `.\pack-all.ps1` |
| 安装模板 | `dotnet new install EfCore.Enterprise.Templates` |
