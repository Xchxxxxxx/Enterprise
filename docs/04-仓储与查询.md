# 第4章：仓储与查询

## 4.1 仓储接口对比

| 接口 | 用途 | 立即获得的方法数 |
|------|------|:---:|
| `IRepository<T>` | 基础 CRUD | 13 |
| `ISuperRepository<T>` | 高级 CRUD + 批量 + 软删除 + 锁定/归档 | 25 |
| `ITreeRepository<T>` | 树形结构专属 | 8 |

## 4.2 注入仓储

```csharp
[Injectable]
public class OrderService : BaseService
{
    private readonly ISuperRepository<Order> _orderRepo;

    public OrderService(ISuperRepository<Order> orderRepo, IUnitOfWork uow, IMapper mapper)
        : base(uow, mapper)
    {
        _orderRepo = orderRepo;
    }
}
```

## 4.3 基础 CRUD

```csharp
// ===== 查询 =====
var order = await _orderRepo.GetByIdAsync(orderId);           // 查单条
var orders = await _orderRepo.GetAllAsync();                  // 查全部
var pending = await _orderRepo.GetListAsync(o => o.Status == OrderStatus.Pending); // 条件查询
var first = await _orderRepo.FirstOrDefaultAsync(o => o.OrderNo == "ORD-001");     // 第一条
var exists = await _orderRepo.AnyAsync(o => o.OrderNo == "ORD-001");               // 是否存在
var count = await _orderRepo.CountAsync(o => o.CustomerId == customerId);          // 计数

// ===== 增删改 =====
var order = new Order("ORD-001", customerId, 99.99m);
await _orderRepo.AddAsync(order);        // 新增
order.Pay();
await _orderRepo.UpdateAsync(order);     // 更新
await _orderRepo.DeleteAsync(orderId);   // 删除（软删除）
await _orderRepo.DeleteAsync(order);     // 删除（软删除，传实体）

// ===== 批量新增 =====
await _orderRepo.AddRangeAsync(orderList);
await _orderRepo.UpdateRangeAsync(orderList);
await _orderRepo.DeleteRangeAsync(orderList);

// ===== 提交 =====
await _orderRepo.SaveChangesAsync();  // 或 await _unitOfWork.SaveChangesAsync();
```

## 4.4 分页查询

### 4.4.1 标准分页

```csharp
var request = new PagedRequest
{
    PageIndex = 1,
    PageSize = 20,
    SortField = "CreatedTime",  // 动态排序字段名
    IsAscending = false         // 降序
};

var result = await _orderRepo.GetPagedAsync(
    o => o.CustomerId == customerId,  // 过滤条件
    request);

// result.Items        // List<Order> 当前页数据
// result.TotalCount   // int 总记录数
// result.PageIndex    // int 当前页码
// result.PageSize     // int 每页大小
// result.TotalPages   // int 总页数
// result.HasPreviousPage  // bool
// result.HasNextPage      // bool
```

### 4.4.2 游标分页（大数据量）

```csharp
// 游标分页比 offset 分页快，适合大数据量
var result = await _orderRepo.GetPagedWithCursorAsync(
    o => o.Status == OrderStatus.Pending,
    lastId: null,    // 第一次传 null，后续传上一页最后一条的 Id
    pageSize: 20);
```

### 4.4.3 动态条件拼接

```csharp
using System.Linq.Expressions;

Expression<Func<Order, bool>> filter = o => true;

if (!string.IsNullOrEmpty(keyword))
    filter = filter.And(o => o.OrderNo.Contains(keyword));

if (status.HasValue)
    filter = filter.And(o => o.Status == status.Value);

if (startTime.HasValue)
    filter = filter.And(o => o.CreatedTime >= startTime.Value);

var orders = await _orderRepo.GetListAsync(filter);
```

## 4.5 批量操作

```csharp
// 批量插入
await _orderRepo.BulkAddAsync(orderList);

// 批量合并（按 key 判断是新增还是更新）
await _orderRepo.BulkMergeAsync(orderList, o => o.OrderNo);

// 批量删除（按条件）
await _orderRepo.BulkDeleteAsync(o => o.Status == OrderStatus.Cancelled);

// 按 ID 列表查询
var orders = await _orderRepo.GetByIdsAsync(new List<long> { 1, 2, 3 });
```

## 4.6 软删除与恢复

```csharp
// 软删除（设置 IsDeleted = true）
await _orderRepo.SoftDeleteAsync(orderId);
await _orderRepo.SoftDeleteAsync(order);           // 传实体
await _orderRepo.SoftDeleteRangeAsync(idList);     // 批量软删除

// 恢复（设置 IsDeleted = false）
await _orderRepo.RestoreAsync(orderId);
```

> 软删除仅对继承 `BaseAuditEntity` 及以上的实体生效。
> `Query()` 方法自动过滤已删除数据（`IsDeleted = false`）。

## 4.7 锁定与归档

```csharp
// 锁定（设置 IsArchived = true）
await _orderRepo.LockAsync(orderId);

// 解锁
await _orderRepo.UnlockAsync(orderId);

// 归档（设置 IsArchived + ComplianceArchiveTime）
await _orderRepo.ArchiveAsync(orderId);

// 批量归档
await _orderRepo.ArchiveRangeAsync(idList);
```

> 锁定和归档仅对继承 `BaseComplianceEntity` 的实体生效。

var exists = await _orderRepo.ExistsByUniqueKeyAsync(

## 4.9 大数据导出

```csharp
// 一次性导出（适合中等数据量）
var data = await _orderRepo.GetExportDataAsync(o => o.CreatedTime >= startDate);

// 流式查询（适合海量数据，避免内存溢出）
await foreach (var order in _orderRepo.StreamQueryAsync(o => o.CreatedTime >= startDate))
{
    // 逐条处理，写入文件或批量发送
    await writer.WriteLineAsync(JsonSerializer.Serialize(order));
}
```

## 4.10 事务

```csharp
// 方式一：使用 UnitOfWork（推荐）
await _unitOfWork.BeginTransactionAsync();
try
{
    var order1 = await _orderRepo.GetByIdAsync(1);
    var order2 = await _orderRepo.GetByIdAsync(2);

    order1!.Pay();
    order2!.Cancel("联动取消");

    await _orderRepo.UpdateAsync(order1);
    await _orderRepo.UpdateAsync(order2);
    await _unitOfWork.SaveChangesAsync();
    await _unitOfWork.CommitTransactionAsync();
}
catch
{
    await _unitOfWork.RollbackTransactionAsync();
    throw;
}

// 方式二：使用 ITransactionManager（分布式事务）
await _transactionManager.ExecuteInTransactionAsync(async () =>
{
    // 跨数据库操作
});
```

## 4.11 仓储方法完整列表

### IRepository<T>

| 方法 | 返回类型 | 说明 |
|------|---------|------|
| `Query()` | `IQueryable<T>` | 获取查询对象（自动过滤软删除） |
| `GetByIdAsync(id)` | `Task<T?>` | 根据ID查询 |
| `GetAllAsync()` | `Task<List<T>>` | 查询全部 |
| `GetListAsync(predicate)` | `Task<List<T>>` | 条件查询 |
| `FirstOrDefaultAsync(predicate)` | `Task<T?>` | 第一条 |
| `AnyAsync(predicate)` | `Task<bool>` | 是否存在 |
| `CountAsync(predicate)` | `Task<int>` | 计数 |
| `GetPagedAsync(predicate, request)` | `Task<PagedResult<T>>` | 分页查询 |
| `AddAsync(entity)` | `Task<T>` | 新增 |
| `AddRangeAsync(entities)` | `Task<List<T>>` | 批量新增 |
| `UpdateAsync(entity)` | `Task` | 更新 |
| `UpdateRangeAsync(entities)` | `Task` | 批量更新 |
| `DeleteAsync(entity)` | `Task` | 删除 |
| `DeleteAsync(id)` | `Task` | 按ID删除 |
| `DeleteRangeAsync(entities)` | `Task` | 批量删除 |
| `SaveChangesAsync()` | `Task<int>` | 提交变更 |

### ISuperRepository<T> 额外方法

| 方法 | 返回类型 | 说明 |
|------|---------|------|
| `GetPagedWithCursorAsync(predicate, lastId, pageSize)` | `Task<PagedResult<T>>` | 游标分页 |
| `GetByIdsAsync(ids)` | `Task<List<T>>` | 按ID列表查询 |
| `BulkAddAsync(entities)` | `Task` | 批量插入（自动提交） |
| `BulkMergeAsync(entities, matchKey)` | `Task` | 批量合并 |
| `BulkDeleteAsync(predicate)` | `Task` | 批量条件删除 |
| `SoftDeleteAsync(id)` | `Task` | 软删除 |
| `SoftDeleteAsync(entity)` | `Task` | 软删除（传实体） |
| `SoftDeleteRangeAsync(ids)` | `Task` | 批量软删除 |
| `RestoreAsync(id)` | `Task` | 恢复软删除 |
| `LockAsync(id)` | `Task` | 锁定 |
| `UnlockAsync(id)` | `Task` | 解锁 |
| `ArchiveAsync(id)` | `Task` | 归档 |
| `ArchiveRangeAsync(ids)` | `Task` | 批量归档 |
| `ExistsByUniqueKeyAsync(predicate)` | `Task<bool>` | 唯一键检查 |
| `GetExportDataAsync(predicate)` | `Task<List<T>>` | 导出数据 |
| `StreamQueryAsync(predicate)` | `IAsyncEnumerable<T>` | 流式查询 |
