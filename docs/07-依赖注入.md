# 第7章：依赖注入

## 7.1 自动注册（一行搞定）

在你的类上加 `[Injectable]`，框架自动扫描并注册到 DI 容器：

```csharp
using EfCore.Enterprise.Shared.DependencyInjection;
using Microsoft.Extensions.DependencyInjection;

// 默认 Scoped 生命周期（最常用）
[Injectable]
public class OrderService : BaseService { }

// 指定 Singleton
[Injectable(ServiceLifetime.Singleton)]
public class IdGenerator { }

// 指定 Transient
[Injectable(ServiceLifetime.Transient)]
public class OrderValidator { }
```

**不需要写 `services.AddScoped<...>()`**，不需要手动注册任何服务。

## 7.2 接口名不匹配时指定 ExposeAs

```csharp
// 类名: MediatorEventBus
// 接口名: IDomainEventBus（不匹配默认约定 IMediatorEventBus）
[Injectable(ServiceLifetime.Scoped, ExposeAs = typeof(IDomainEventBus))]
public class MediatorEventBus : IDomainEventBus { }

// 类名: UserService
// 接口名: IUserService（自动匹配，不需要 ExposeAs）
[Injectable]
public class UserService : IUserService { }
```

**默认约定**：框架自动匹配 `I{ClassName}` 接口。比如 `OrderService` 自动注册为 `IOrderService`。

## 7.3 排除某些类

```csharp
using EfCore.Enterprise.Shared.DependencyInjection;

[IgnoreInjectable]
public class InternalHelper
{
    // 不会被自动注册
}
```

## 7.4 生命周期选择指南

| 生命周期 | 创建时机 | 释放时机 | 适用场景 |
|---------|---------|---------|---------|
| `Singleton` | 应用启动时创建一次 | 应用关闭时 | 无状态工具类、ID生成器、配置 |
| `Scoped` | 每个请求创建一个 | 请求结束时 | **仓储、Service、领域服务（最常用）** |
| `Transient` | 每次注入都创建新的 | 使用完毕后 | 轻量级无状态服务、验证器 |

## 7.5 构造函数注入

```csharp
[Injectable]
public class OrderService : BaseService
{
    private readonly ISuperRepository<Order> _orderRepo;
    private readonly ICacheService _cache;
    private readonly IMapper _mapper;
    private readonly IOrderDomainService _domainService;

    // 构造函数注入，.NET 标准方式
    public OrderService(
        ISuperRepository<Order> orderRepo,
        ICacheService cache,
        IMapper mapper,
        IOrderDomainService domainService)
    {
        _orderRepo = orderRepo;
        _cache = cache;
        _mapper = mapper;
        _domainService = domainService;
    }
}
```

## 7.6 模块化注册（IModule）

批量注册相关服务，适合复杂初始化逻辑：

```csharp
using EfCore.Enterprise.Shared.DependencyInjection;
using Microsoft.Extensions.Configuration;
using Microsoft.Extensions.DependencyInjection;

public class PaymentModule : IModule
{
    public void ConfigureServices(IServiceCollection services, IConfiguration configuration)
    {
        services.AddHttpClient<IPaymentGateway, AlipayGateway>();
        services.Configure<PaymentOptions>(configuration.GetSection("Payment"));
        services.AddScoped<IPaymentService, PaymentService>();
    }
}
```

在 `Program.cs` 中注册：

```csharp
builder.Services.AddModules(builder.Configuration, typeof(PaymentModule).Assembly);
```

## 7.8 完整示例：从标记到使用的全流程

```csharp
// 1. 定义接口
public interface IReportService
{
    Task<byte[]> GenerateMonthlyReportAsync(int year, int month);
}

// 2. 实现 + 标记（一步到位）
[Injectable]
public class ReportService : IReportService
{
    private readonly ISuperRepository<Order> _orderRepo;

    // 3. 构造函数注入依赖
    public ReportService(ISuperRepository<Order> orderRepo)
    {
        _orderRepo = orderRepo;
    }

    public async Task<byte[]> GenerateMonthlyReportAsync(int year, int month)
    {
        // 业务逻辑
        var orders = await _orderRepo.GetListAsync(o =>
            o.CreatedTime.Year == year && o.CreatedTime.Month == month);
        // 生成报告...
        return Array.Empty<byte>();
    }
}

// 4. 在任何地方注入使用
[Injectable]
public class ReportController : ControllerBase
{
    private readonly IReportService _reportService;

    public ReportController(IReportService reportService)
    {
        _reportService = reportService;
    }

    [HttpGet("monthly/{year}/{month}")]
    public async Task<IActionResult> GetMonthlyReport(int year, int month)
    {
        var report = await _reportService.GenerateMonthlyReportAsync(year, month);
        return File(report, "application/pdf");
    }
}
```
